Tabela de Medidas CSS, HTML e JavaScript

Medida Descrição Uso Comum
px (pixels) Unidade de medida que representa um único ponto na tela do dispositivo. Utilizada para definir tamanhos exatos, como largura e altura de elementos.
em Unidade relativa que é baseada no tamanho da fonte do elemento pai. 1em é igual ao tamanho da fonte do elemento pai. Comumente usada para definir tamanhos de fonte e espaçamentos que são relativos ao tamanho do texto.
rem (root em) Unidade relativa que é baseada no tamanho da fonte do elemento raiz (html). 1rem é igual ao tamanho da fonte definida no elemento html. Utilizada para criar layouts consistentes que são independentes do contexto de aninhamento.
% (porcentagem) Unidade relativa que representa uma porcentagem do tamanho do elemento pai. Utilizada para definir tamanhos fluidos que se ajustam ao tamanho do elemento pai.
vh (viewport height) Unidade relativa que representa 1% da altura da viewport (área visível do navegador). Utilizada para definir tamanhos que se ajustam à altura da janela do navegador.
vw (viewport width) Unidade relativa que representa 1% da largura da viewport (área visível do navegador). Utilizada para definir tamanhos que se ajustam à largura da janela do navegador.
vmin Unidade relativa que representa 1% do menor valor entre a altura e a largura da viewport. Utilizada para definir tamanhos que são proporcionais à menor dimensão da viewport.
vmax Unidade relativa que representa 1% do maior valor entre a altura e a largura da viewport. Utilizada para definir tamanhos que são proporcionais à maior dimensão da viewport.
pt (points) Unidade de medida tradicionalmente usada na impressão. 1pt é igual a 1/72 de uma polegada. Menos comum na web, mas pode ser usada para definir tamanhos de texto em PDFs e outros documentos impressos.
cm (centímetros) Unidade de medida que representa um centímetro real. 1cm é igual a 37.8 pixels. Utilizada para definir tamanhos físicos de elementos, principalmente em impressão.
mm (milímetros) Unidade de medida que representa um milímetro real. 1mm é igual a 3.78 pixels. Utilizada para definir tamanhos físicos de elementos, principalmente em impressão.
in (inches) Unidade de medida que representa uma polegada real. 1in é igual a 96 pixels. Utilizada para definir tamanhos físicos de elementos, principalmente em impressão.
